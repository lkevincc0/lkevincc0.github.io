Markdown Cheat Sheet✔︎

Liu published on 2024-10-02

This is a complete Markdown example file showcasing all common Markdown syntax usages, ✍️.

1. Headings

You can use different numbers of # to indicate the level of headings:

H1 Heading

H2 Heading

H3 Heading

H4 Heading

H5 Heading
H6 Heading

2. Emphasizing Text

Bold is achieved by wrapping the text with two asterisks **.

Italic is achieved by wrapping the text with one asterisk *.

Bold and Italic is achieved by wrapping the text with three asterisks ***.
